# OrchestrAI - Automatización con IA

Proyecto web para OrchestrAI, consultoría de automatización de procesos con inteligencia artificial.

## 🚀 Despliegue en Vercel

### Instalación
```bash
npm install -g vercel
```

### Despliegue
```bash
vercel --prod
```

## 📁 Estructura de Archivos

- `web_operis_completa.html` - Página principal
- `politica-privacidad.html` - Política de privacidad
- `aviso-legal.html` - Aviso legal
- `politica-cookies.html` - Política de cookies
- `favicon.svg` - Icono del sitio
- `vercel.json` - Configuración de Vercel

## 🔧 Características

- ✅ Captación de leads con Google Sheets
